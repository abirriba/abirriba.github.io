Le développement d'un jeu vidéo utilise plusieurs langages de programmations. Les premiers jeux étaient développés en langage assembleur, version humaine du langage machine, puis, permettant un développement plus rapide, vers des langages plus structuré lors de leurs apparitions. Le langage assembleur est cependant toujours utilisé pour les tâches nécessitant une très grande optimisation. Sur les premiers micro-ordinateurs, le BASIC était le langage le plus répandu, puis, il a été progressivement supplanté par le langage C, venant des systèmes Unix et permettant une approche plus bas niveau. Enfin, il existe aujourd'hui de nombreux moteurs de jeux, qui s'occupent de ses couches bas niveau. Ceux-ci peuvent être des API utilisés dans des langages de programmation (C, C++, etc), ou langage de script (JavaScript, lua, Python, Ruby, etc.).
